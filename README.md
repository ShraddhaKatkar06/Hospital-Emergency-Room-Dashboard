# Hospital Emergency Room Dashboard (Excel Project)

## Project Overview
This project is an interactive **Hospital Emergency Room Dashboard** built using **Microsoft Excel** to analyze hospital patient data and generate meaningful healthcare insights.

The dashboard helps track **patient flow, waiting time, patient satisfaction, admission status, and department referrals** using KPI-driven analysis and visualizations.

---

## Project Objective
The objective of this dashboard is to analyze emergency room performance and improve healthcare decision-making by monitoring important patient-related metrics.

---

## Key Performance Indicators (KPIs)

The dashboard tracks the following KPIs:

- **Number of Patients** – Total patients visiting the emergency room.
- **Average Wait Time** – Average time patients wait to see a medical professional.
- **Patient Satisfaction Score** – Measures service quality based on patient feedback.

---

## Dashboard Features

### 1. Patient Admission Status
- Compare admitted vs non-admitted patients.
- Analyze hospital admission trends.

### 2. Patient Age Distribution
- Group patients into different age categories.
- Identify age groups with the highest ER visits.

### 3. Timeliness Analysis
- Measure the percentage of patients attended on time vs delayed.

### 4. Gender-wise Analysis
- Analyze patient distribution by male and female.

### 5. Department Referrals
- Identify departments receiving the highest patient referrals.
